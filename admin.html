Deployment Notes

Open deployment checklist
Frontend

Deploy the static site on Vercel and set the FRONTEND_API_BASE environment variable to your Render API URL.

Backend

Deploy the Express API on Render from the server directory and use /api/health/live as the health check path.

Database

Connect the backend to MongoDB using MONGODB_URI and keep JWT_SECRET out of source control.

Storage

Attach a persistent disk on Render so uploads and approved papers survive redeploys.

Payments

Set the M-Pesa callback URL to /api/subscriptions/mpesa/callback on your Render API domain.
