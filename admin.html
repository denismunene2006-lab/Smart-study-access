Deployment Notes

Open deployment checklist
Frontend

Deploy the static site on Vercel and set the FRONTEND_API_BASE environment variable to your hosted Node API URL.

Backend

Deploy the Express API on Render / Railway from the server directory and use /api/health/live as the health check path.

Database

Execute server/supabase-schema.sql in the Supabase SQL editor. Set SUPABASE_URL, SUPABASE_ANON_KEY, and SUPABASE_SERVICE_ROLE_KEY environment variables on your API host.

Storage

Create two Storage buckets in the Supabase dashboard named papers and uploads with appropriate access policies.

Payments

Configure your M-Pesa sandbox credentials. Set the callback URL to point to /api/subscriptions/mpesa/callback on your API host domain.
